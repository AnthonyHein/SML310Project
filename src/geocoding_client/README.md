# Geocoding Client

This is a Python client for the API provided by Google Maps. Documentation about this API can be found at the following links:

- [https://developers.google.com/maps/documentation/geocoding/overview](https://developers.google.com/maps/documentation/geocoding/overview)
- [https://developers.google.com/maps/documentation/geocoding/get-api-key#console](https://developers.google.com/maps/documentation/geocoding/get-api-key#console)
